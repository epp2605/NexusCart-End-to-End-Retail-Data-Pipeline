# NexusCart – Retail Sales Analytics Pipeline

## Project Summary
NexusCart is a retail sales analytics project designed to analyze customer transactions, product performance, and sales trends using Python, SQL, and Advanced Excel. The project demonstrates data cleaning, analysis, visualization, and customer segmentation techniques commonly used in Data Analytics.

## Business Objectives
- Analyze retail sales performance across products.
- Identify top revenue-generating products.
- Evaluate customer spending patterns.
- Track sales trends over time.
- Segment customers based on purchasing behavior.

## Technology Stack
- Python (Pandas, Matplotlib)
- SQL
- Advanced Excel (XLOOKUP, Pivot Tables, Pivot Charts)
- Git & GitHub

## Analytics Performed

### Excel Analysis
- Created Product Master dataset.
- Applied XLOOKUP for product mapping.
- Built Pivot Tables for revenue analysis.
- Created Pivot Charts for business reporting.

### SQL Analysis
- Wrote analytical SQL queries to calculate:
  - Product-wise revenue
  - Customer-wise revenue
  - Sales performance metrics

### Python Analysis
- Performed RFM (Recency, Frequency, Monetary) Analysis.
- Segmented customers into:
  - VIP Customers
  - Regular Customers
  - At-Risk Customers
- Built visual dashboards using Matplotlib.

## Key Insights
- Product P002 generated the highest revenue.
- Customer spending varies significantly across segments.
- Revenue trends help identify high-performing periods.
- RFM analysis highlights valuable and at-risk customers.
